If you don't use braces ({}) after control flow statements (like if, else, for, while, etc.), then only the immediately following statement is considered to be inside the control flow block.

if (condition) 
DoSomething();
DoSomethingElse(); 

In this case, DoSomething() will be executed if the condition is true, while DoSomethingElse() will always be executed regardless of the condition, because it's not inside the if block. Even if identation is applied this wont work.
